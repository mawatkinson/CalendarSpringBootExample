. Clone git repository 
. Install and configure a blank mySQL database.
. Execute 'calendar\src\main\resources\SQL database schema setup.sql' against new database.
. Update 'calendar\src\main\resources\application.properties' database url, username and password.
. Execute maven goal 'spring-boot:run' to deploy the application.
. Consult 'calendar\src\main\resources\REST endpoints.txt' for example happy path REST calls
